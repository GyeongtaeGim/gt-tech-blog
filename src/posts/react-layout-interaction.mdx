framer motion을 이용한 layout interaction 예제 입니다.

json server를 이용하서 api를 mocking 했고 카드셔플과 스켈레톤, 레이아웃 인터렉션을 적용했습니다. 필터와 카드 클릭에 따라서 자연스로운 동작이 수행됩니다.

레이아웃 인터렉션은 곧 웹 생태계에서 view-transition이 보편화되면 더 고도화되길 기대하고 있습니다.
